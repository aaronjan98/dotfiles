# Zettelkasten Search

## Purpose
Search AJ's zettelkasten for notes relevant to the current task.

## Use this when
- A project declares the zettelkasten as a dependency
- Conceptual notes, source notes, or durable notes may help
- The agent needs supporting context beyond the current repository

## Inputs
- topic or question
- preferred folders if known
- whether to prioritize stable notes or exploratory notes

## Search guidance
- Prefer `Zettels/` for stable knowledge
- Use `Outside/` for source-faithful note capture
- Use `Inside/` cautiously because it may contain exploratory thinking
- Use `Catalog/` for navigation only when helpful

## Output
Return:
1. the most relevant note paths
2. a short reason each note is relevant
3. any uncertainty about note quality or stability
