# Skills Directory

Reusable procedures for common tasks. Each subdirectory contains a `SKILL.md` with full instructions.

When a user's request matches a skill below, read that skill's `SKILL.md` and follow it rather than improvising.

## Available skills

| Skill | Folder | Domain | Use when the user says... |
|-------|--------|--------|--------------------------|
| Lecture Notes Merge | `lecture-notes-merge/` | School | "merge my notes with the slides", "fill in my notes from the PDF", "integrate the lecture slides into my notes" |
| Academic Book Extraction | `academic-book-extraction/` | School / research | "extract notes from this reading", "summarize this chapter", "pull key ideas from this PDF" |
| Jupyter Notebook Inspection | `jupyter-notebook-inspection/` | Scientific computing | "inspect this notebook", "what does this notebook do", "summarize this Jupyter file" |
| Zettelkasten Search | `zettelkasten-search/` | Personal knowledge | "search my notes", "find notes about X", "look in my zettelkasten" |
| Zettelkasten New Directory | `zettelkasten-new-directory/` | Personal knowledge | creating a new folder or directory inside the zettelkasten vault |
| Save Session | `save-session/` | Meta | "save the session", "wrap up", "end of session" |
