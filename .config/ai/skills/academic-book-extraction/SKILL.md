# Academic Book Extraction

## Purpose
Extract structured academic notes from a book chapter, PDF, or assigned reading.

## Use this when
- A project depends on course readings, books, papers, or PDFs
- The agent needs structured notes rather than raw excerpts
- The material is part of academic research or coursework

## Inputs
- file path
- reading title if known
- current research question or task
- optional section or chapter focus

## Output
Produce:
1. citation or source identification
2. key claims
3. important definitions
4. evidence or examples
5. open questions
6. relevance to the current project

## Rules
- Prefer faithful extraction over premature interpretation
- Separate the author's claims from the agent's synthesis
- Note when confidence is limited because the source was only partially reviewed
