# Memory Model

## Project operational memory
Operational notes live inside the active project in:

- `memory/YYYY-MM-DD.md`

These notes record:
- what was done today
- blockers
- next steps
- transient discoveries

## Project durable memory
Durable project knowledge lives inside the active project in:

- `project-memory/`

This should contain stable facts such as:
- accepted terminology
- key decisions
- stable structure choices
- durable project truths

## Global shared memory
Global, reusable AI conventions live in `~/.config/ai/shared/`.

These are not project-specific facts.
