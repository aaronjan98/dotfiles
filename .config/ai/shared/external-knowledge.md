# External Knowledge Sources

External knowledge sources are not part of the default context for every project.

They should only be used when a project's `DEPENDENCIES.md` declares them relevant.

## Known external knowledge roots
- `~/Repositories/self-hosted/zettelkasten`
- `~/Documents/School`

## Zettelkasten guidance
Different parts of the vault have different roles:

- `Zettels/` contains more stable knowledge notes.
- `Outside/` contains notes from external sources.
- `Inside/` contains exploratory or in-progress thought.
- `Catalog/` is navigation and overview.
- `Documents/` contains assets and supporting materials.

Projects should state which parts are most useful for that project.
