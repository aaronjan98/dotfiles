# 2026-04-19 — MCP setup

## What changed
- added `context7` MCP configuration to `opencode.json` as a remote MCP
- added `playwright` MCP configuration to `opencode.json` as a local MCP started with `npx`
- documented that `CONTEXT7_API_KEY` should come from NixOS-managed runtime environment, not from tracked OpenCode config

## Important notes
- Context7 uses `CONTEXT7_API_KEY` for higher rate limits
- Playwright MCP does not require an API key for local use
- both MCPs are meant to be reproducible through tracked config plus NixOS-managed secrets
